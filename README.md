# findr

An open source search engine experiment.

## Overview

This project is an experimental implementation of a search engine built from scratch. It serves as a learning tool to understand the core concepts and architecture of search engine systems, including web crawling, indexing, and retrieval mechanisms.

## About

findr is primarily a personal learning project to explore how search engines work. The codebase contains implementations for web spidering, indexing, and search functionality written in Python with a web interface.
